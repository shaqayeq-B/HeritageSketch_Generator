# ArchaeoSketch 🏛️

Convert images of historical and architectural buildings into clean sketch-style line drawings using OpenCV.

این پروژه برای تبدیل تصاویر بناهای تاریخی و معماری به اسکچ خطی طراحی شده است.  
می‌تواند برای مستندسازی، تحلیل بصری، یا آماده‌سازی تصاویر پژوهشی در حوزه معماری و باستان‌شناسی استفاده شود.

---

## ✨ Features

- حذف نویز و بافت‌های ناخواسته دیوار
- تقویت کنتراست برای جداسازی عناصر معماری
- استخراج خطوط ساختاری با Adaptive Threshold
- تمیزکاری مورفولوژیک برای حذف نویزهای ریز
- خروجی اسکچ آماده ذخیره

---

## 📦 Requirements

Python 3.8+

کتابخانه‌های مورد نیاز:
opencv-python numpy matplotlib
همچنین تصویر در محیط اجرا نمایش داده می‌شود.
